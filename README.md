# AdventOfCode2025.FSharp

How to use repo: 
1. Make sure F# is installed in the Visual studio installer.
2. Open .sln in Visual Studio 2026+. 
3. Set _AdventOfCode2025.FSharp_ as starting project and run/debug to develop new puzzle solutions.

How to run one of the existing solutions:
1. Set _AdventOfCode2025.FSharp.Finished_ as starting project. 
2. Uncomment the res1/res2 lines and modify the Day as needed. 
3. Modify the relevant _input##.txt_ with your input. 
4. Run/debug
